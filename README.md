Aplikasi Rak Buku Digital - Kelola koleksi bacaanmu dengan mudah!

Bookshelf App adalah aplikasi web interaktif untuk mengelola daftar buku pribadi. Dibangun dengan JavaScript murni (Vanilla JS) dan desain modern yang responsif. Aplikasi ini memungkinkan pengguna untuk menambah, mencari, mengedit, menghapus, serta memindahkan buku antar rak "Belum selesai dibaca" dan "Selesai dibaca" dengan penyimpanan data menggunakan Local Storage.
